# siac-ui

Application UI developed using Angular.

## Features

* Student Login
* Registration
* Forgot Password
* Application Form
* Edit Application
* Payment Module
* Application Preview
* Report Generation

## Technologies Used

* Angular 21
* TypeScript
* Playwright
* GitHub Actions

## Setup

### Node Version

* Node.js 20.19.0
* npm 10.8.2

### Start Application

```bash
ng serve
```

Open:

```
http://localhost:4200
```

## Build

```bash
ng build
```

## Run Unit Tests

```bash
ng test
```

## Run Playwright Tests

```bash
npx playwright test
```

## Project Structure

* Login
